## prerequisites


In the terminal...


1. Python3:

`$ Python3 --version`


if not found:


`$ brew install python3`


If you don't have brew:


`/usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"`



2. Packages:


`pip install matplotlib`

`pip install numpy`
